I've been experimenting with tiled desktops lately, and after configuring them a bit I feel like the experience is pretty good

## 1.i3
直接apt install i3即可

## 2. Wallpaper configuration
When switching into i3wm, the whole page is a black blob.
I use the wallpaper configured by Nitrogen.
Just add a self-starter to the i3 configuration
``
exec --no-startup-id nitrogen --restore
```
BTW, the wallpaper source is https://artvee.com/

## volume
If you want to use the volume and brightness keys on your laptop to adjust them, you'll also need to configure them
For volume I use pamixer

The i3 configuration:
``
# Increase volume.

bindsym XF86AudioRaiseVolume exec --no-startup-id pamixer -i 10 && notify-send "音量" "$(pamixer --get-volume)%"
 
# 降低音量
bindsym XF86AudioLowerVolume exec --no-startup-id pamixer -d 10 && notify-send "音量" "$(pamixer --get-volume)%"

# Mute
bindsym XF86AudioMute exec --no-startup-id pamixer -t && notify-send "volume" "$(pamixer --get-mute | grep -q true && echo 'muted' || echo 'unmuted')"
``

## Brightness
Brightness using: brightnessctl

brightnessctl requires sudo privileges.
First, configure a password-free
``
sudo visudo -f /etc/sudoers.d/brightnessctl
```
Add:
```
finn ALL=(ALL) NOPASSWD: /usr/bin/brightnessctl
```
Save and exit, then write in the i3 configuration file again
```
bindsym XF86MonBrightnessUp exec --no-startup-id "sudo brightnessctl set +5% | grep -oP '\\d+%' | xargs -I {} notify-send 'Brightness increasing' 'Current brightness: {}'"

bindsym XF86MonBrightnessDown exec --no-startup-id "sudo brightnessctl set 5%- | grep -oP '\\d+%' | xargs -I {} notify-send '亮度降低' '当前亮度: {}'"
```
